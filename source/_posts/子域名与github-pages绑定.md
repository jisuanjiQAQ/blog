# 将子域名解析到GitHub

进入域名管理后台

新建一个dns解析→类型为CNAME→记录值为用户名.github.io→名称为你想解析的子域名

# 仓库绑定

在仓库的Setting->Pages里，在自定义域名里输入 DNS 配置好的域名，点击保存，GitHub 会自动进行 DNS 校验，这个过程需要一点时间

当校验通过以后，就可以勾选下面的强制开启 HTTPS ，是不需要申请 SSL 证书的，GitHub 会自动帮申请。~~免费万岁~~
